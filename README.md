# X-Growth · 海外"离谱搞钱"案例（蠢生意流 · 中文信息差套利）

**一句话定位**：我帮中国人扒那些"卧槽这也行"的海外离谱搞钱案例——真实、好笑、有数字、有来源。

**招牌**：**蠢生意 / 反常案例**（给仇人寄闪粉那种），用"这也行？"的猎奇感引流。

**模式**：把 Indie Hackers / Starter Story / Reddit / 奇葩产品报道里的离谱真实案例，**汉化 + 猎奇拆解 + 行动呼吁**，发到 **中文 X + 小红书**。

**目标**：90 天内 X 达成创作者收入门槛、小红书破千粉接品牌合作，并通过私域知识付费 + 联盟 + 品牌合作实现月入副业收入。

## 为什么是这个模式

- **不需要你自己的成功案例**——你扒的是别人的真实离谱案例（带来源、带数字）
- **护城河 = 信息差 + 猎奇**：中国受众看不到这些一手源，而"离谱/好笑"自带传播
- **不虚**：越离谱越要标来源、给数字（可信度三要素）
- **双平台**：中文 X（thread）+ 小红书（图文，涨粉最快的中文平台）

## 内容三柱 (60 / 25 / 15)

| 柱 | 占比 | 类型 | 流量逻辑 |
|---|---|---|---|
| **WTF 猎奇案例** | 60% | 一个离谱/反常/自带传播的真实赚钱案例（蠢生意/卖铲子/暴起暴落/窥私/极限荒诞） | "卧槽这也行"，自带转发 |
| **LIST 信息差清单** | 25% | "中文圈没人讲的 N 种赚钱方式"清单体 | 高收藏，一条钩出多个未来选题 |
| **HOOK 情绪共鸣** | 15% | 经济焦虑 / 打工痛点 / 副业共鸣 | 情绪共鸣，流量入口 |

## 猎奇雷达（选题第一原则）

不扒"成功人士做对三件事"的教科书案例。专挑能让人"**卧槽这也行？**"的：
🤪 蠢生意（招牌）· ⛏️ 卖铲子 · 💥 暴起暴落 · 🕵️ 窥私账本 · 🚇 地下赛道 · ⚡ 极限荒诞

## 写法铁律（能不能爆的关键）

1. **好奇心缺口**：每条留钩子勾下一条，绝不一条说完
2. **必有"反转"**：每篇挖出一个"最骚的一步"放中后段引爆
3. **口语去翻译腔**：像八卦，不像报告
4. **戏剧 > 道理**：先故事和反差，认知点最后轻点
5. **真实兜底**：可信度三要素 = ① 具体数字 ② 原始来源链接 ③ 时间线

## 变现路径（中文优先级）

1. **引流私域（微信）→ 知识付费 / 付费社群**（中文最强变现）
2. **小红书品牌合作**（蒲公英，需 1000 粉 + 实名）
3. **信息产品**：海外案例库 / prompt 包 / "海外搞钱周报"（知识星球 / 小报童）
4. **联盟**：海外 AI 工具 + 出海工具返佣
5. **X Ads Revenue Share**（达 5M impression 后，中文 CPM 偏低，次要）

## 红线

1. **拆解≠抄袭**：每篇标注原始来源（Source: Starter Story / @levelsio），是"我帮你读了英文" 不是"我编的"
2. **不编造 / 不夸大数字**——以原案例披露为准
3. **不荐股、不荐币、不做投资建议**（合规）
4. AI 生成内容按平台规则标注
5. 不买粉、不刷量
6. 不碰政治
