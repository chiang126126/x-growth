# Prompt · Thread 起草

**用途**：把信号过滤器选中的选题扩写成完整 7-12 条 Thread 草稿。

**推荐模型**：Claude Opus 4.8（叙事最强）或 GPT-5（hook 更狠）

---

## Prompt 本体

```
You're drafting a Thread for an English X account about making money
with AI in a tough economy. Our edge is PROOF — real numbers, real
prompts, real screenshots. Voice: a sharp friend who actually does
this and shows the receipts, NOT a guru selling a dream. We use
specific dollar amounts, real tool names, and one concrete,
copyable step per tweet.

TOPIC: [paste title from signal filter]
PILLAR: [PROOF / PLAYBOOK / HOOK]
OUTLINE: [paste the 5-7 bullet outline]
TOOLS/AFFILIATE: [paste the tools that fit]
MY REAL DATA: [paste any real numbers/screenshots I have, e.g.
  "agent earned $217 last week across 3 gigs" — leave blank if none]

Draft a Thread of 7-12 tweets following these rules:

TWEET 1 (Hook) — must use this structure:
  - Line 1: a proof-receipt or number-shock or contrarian opener
    (e.g., "I made $217 last week with an AI agent I built in an
    afternoon. No course, no BS. Here's the exact setup:")
  - Line 2: promise of concrete payoff ("8 tweets, 1 prompt, and
    the actual tool I used")
  - <280 chars total

TWEETS 2 to N-1 (Body):
  - Each tweet stands alone (could be screenshotted)
  - Each has ONE concrete, actionable thing (a step, a number, a
    prompt, a tool, a screenshot placeholder)
  - When giving a prompt, write it out in full so people can copy it
  - No filler ("It's important to note", "Furthermore")
  - Use line breaks aggressively
  - Mark screenshots with [IMG: ...] (real screenshot, never AI image)

TWEET N (CTA):
  - 1-line recap
  - One clear ask: "RT the first tweet if useful" OR "I send the
    full breakdown + prompts every Sunday — link in bio"
  - If a tool fits, mention it honestly with #ad and note it's
    affiliate

CONSTRAINTS:
  - NO fabricated income numbers. If MY REAL DATA is blank, frame
    as "here's the setup" not "here's what I earned." Honesty is
    the whole brand.
  - No get-rich-quick / "passive $10k/mo" language
  - No investment / stock / crypto advice
  - No em-dashes (use ":" or "."), no "🧵" in tweet 1, no emoji spam
  - Mark every factual claim with [src: domain.com] for me to verify
  - Disclose AI assistance only if the output text itself is what's
    being published as AI-generated

Output format:

  ---TWEET 1---
  [content]

  ---TWEET 2---
  [content]
  [IMG: description]

  ... etc

  ---NOTES---
  - Verification checklist (claims/numbers to double-check)
  - Suggested posting time (Western audience, EST/PST)
  - Affiliate opportunity + suggested disclosure
  - What real screenshot/proof I should attach to make this land
```

---

## 用完之后的人工动作

1. **必须人工重写 Tweet 1**（AI 的 hook 永远不如你改一次）
2. **核对所有数字**——尤其收入数字，必须真实可截图，否则删掉/改框架
3. **截图**——标了 [IMG] 的地方亲自抓真图
4. 念出声读一遍，改掉别扭句子
5. 排版进 Typefully，预览，定时

## 不要做的事
- ❌ 让 AI 编收入数字
- ❌ 一次写 5 个 Thread（质量塌）
- ❌ 完全照搬 AI 输出（算法识别得出）
- ❌ 用 AI 生成图当配图（用真截图）
