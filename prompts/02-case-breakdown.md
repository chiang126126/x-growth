# Prompt · 案例拆解（猎奇写法 · X thread + 小红书图文）

**用途**：把一个"离谱搞钱"案例，一次产出 ① 中文 X thread ② 小红书图文。

**推荐模型**：Claude Opus 4.8

---

## 写法铁律（这是这个号能不能爆的关键）

1. **好奇心缺口**：每条结尾留钩子，让人想点开下一条（"但真正离谱的在后面""问题来了：它凭什么火？"）。绝不一条把话说完。
2. **口语、有人味**：像跟朋友八卦，不是写报告。短句、换行、带点情绪（哈哈、卧槽、你没看错）。
3. **去翻译腔**：禁止"令人震惊的是""值得注意的是"。要像中国人聊天。
4. **必须有"反转"**：每篇要挖出一个"最骚的一步"放在中后段引爆。
5. **戏剧 > 道理**：先讲故事和反差，认知点最后再轻轻点一句，别说教。
6. **真实兜底**：越离谱越要标来源、给数字。猎奇是钩子，真实是信任。

---

## Prompt 本体

```
你在为一个"扒海外离谱搞钱案例"的中文账号写内容（X + 小红书双发）。
招牌是蠢生意/反常案例，风格是"卧槽这也行"的猎奇八卦感。
你是一个读得懂英文、爱扒怪案例讲给朋友听的人。真实、有数字、有来源，
但讲得像八卦一样上头。绝不用"躺赚/暴富/财富自由"。

案例信息：
标题：[粘贴信号过滤器给的标题]
猎奇类型：[蠢生意/卖铲子/暴起暴落/窥私/地下赛道/极限荒诞]
柱：[WTF / LIST / HOOK]
原始来源：[平台 + 链接 + 作者]
核心数字：[收入/售价/时间/起点]
最骚的一步/反转点：[粘贴信号过滤器挖出的反转]
提纲：[粘贴故事化提纲]

请产出两份内容：

=== 一、X 中文 Thread（8-12 条）===

写作要求（强制）：
- 每条结尾留好奇缺口，勾下一条（除了最后一条）
- 口语、短句、多换行，可带轻微情绪词（你没看错/问题来了/最骚的是）
- 禁止教科书式"第1步第2步"罗列；用讲故事的方式推进
- 把"最骚的一步/反转"放在第 5-7 条引爆，前面铺悬念
- 每条尽量一个具体点：数字/反差/细节
- 标 [图：建议放什么真实截图]（绝不用AI生图）

结构（按这个节奏，但要写成连贯故事）：
  第1条 钩子：先抛最离谱的事实+数字，再留悬念（"这逻辑比你想的精明↓"）
  第2-4条：他是谁多离谱 / 这玩意凭什么火（埋"它击中了什么"）
  第5-7条：揭秘玩法 + 引爆"最骚的一步/反转"
  倒数第2条：3条可抄的认知（轻点，不说教）
  最后1条：行动呼吁 + 关注引导 + 标来源

=== 二、小红书图文笔记 ===
  - 封面大标题（≤18字，"这也行？"反差感，可1个emoji）
  - 副标题（≤20字）
  - 正文（300-500字，开头先抛离谱钩子，分点，多换行+emoji）
  - 9张图建议（封面 + 反差对比 + 数字截图 + 来源截图 + 金句卡）
  - 结尾引导（关注/收藏/评论）+ 一句来源标注
  - 5-8个话题标签（埋SEO：#海外搞钱 #离谱生意 #副业 #搞钱思路 等）

=== 三、备注 ===
  - 核实清单：哪些数字/事实必须回原文核对
  - 建议发布时间
  - 该配的真实截图清单
  - 合规检查：来源是否标注、有无夸大、有无投资建议

约束：数字以原案例为准不得编造；必须标来源；不出现荐股荐币；
不用"🧵"开头；不堆感叹号。
```

---

## 用完之后的人工动作
1. **人工重写第1条 / 封面标题**（钩子最值得你亲手打磨）
2. **回原文核对所有数字**——越离谱越要核实
3. **抓真实截图**（收入截图、来源页面截图）
4. 念一遍，删掉翻译腔和说教句
5. X 进 Typefully；小红书图进 Canva/稿定

## 不要做的事
- ❌ 编/夸数字、不标来源
- ❌ 教科书罗列、说教
- ❌ 一条把话说完（没缺口=没人往下看）
- ❌ 翻译腔、AI 生成图当配图
- ❌ 一次让 AI 写 5 篇（质量塌）
